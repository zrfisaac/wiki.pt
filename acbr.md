# ACBr

## Descrição
ACBr (Automação Comercial Brasil) é um conjunto de componentes de código aberto para Delphi e Lazarus, criado para facilitar o desenvolvimento de sistemas comerciais no Brasil. Ele oferece suporte completo a emissão de documentos fiscais eletrônicos, como NF-e, NFC-e, SAT, MDFe, CT-e, além de boletos bancários, TEF (Transferência Eletrônica de Fundos), leitores de código de barras, balanças, impressoras fiscais e muito mais.
