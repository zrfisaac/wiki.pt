# macOS

macOS é o sistema operacional desenvolvido pela Apple para seus computadores Mac. Ele é conhecido por sua interface elegante, desempenho estável e integração com o ecossistema da Apple, incluindo dispositivos como iPhone, iPad e Apple Watch.

## Índice

---

### Descrição de Cada Diretório no macOS

| Diretório              | Descrição                                                                                 |
|------------------------|-------------------------------------------------------------------------------------------|
| `/`                   | Diretório raiz do sistema de arquivos, de onde todos os outros diretórios se ramificam.    |
| `/Applications`        | Contém os aplicativos instalados, geralmente acessados via Finder ou Launchpad.           |
| `/System`              | Abriga arquivos do sistema macOS e componentes essenciais, incluindo o kernel e bibliotecas. |
| `/Library`             | Armazena recursos de uso geral no sistema, como preferências, fontes e arquivos de suporte. |
| `/Users`               | Contém os diretórios pessoais de todos os usuários do sistema.                            |
| `/Users/Shared`        | Um diretório compartilhado acessível por todos os usuários, usado para colaboração.        |
| `/private`             | Armazena arquivos do sistema que não são diretamente visíveis no Finder, como logs e arquivos temporários. |
| `/private/tmp`         | Diretório para arquivos temporários, limpo automaticamente na inicialização.              |
| `/private/var`         | Contém dados variáveis como logs, caches e informações de execução do sistema.            |
| `/Volumes`             | Ponto de montagem para todos os discos externos e unidades de rede conectados ao sistema. |
| `/bin`                 | Binários essenciais para comandos e utilitários da linha de comando.                      |
| `/sbin`                | Binários do sistema usados para tarefas administrativas, geralmente requerem permissões elevadas. |
| `/usr`                 | Contém binários, bibliotecas e recursos compartilhados para aplicativos e ferramentas do sistema. |
| `/usr/bin`             | Utilitários e executáveis de linha de comando disponíveis para todos os usuários.          |
| `/usr/local`           | Destinado a softwares instalados localmente e configurações personalizadas.               |
| `/etc`                 | Armazena arquivos de configuração do sistema para linha de comando e serviços.            |
| `/dev`                 | Arquivos especiais que representam dispositivos de hardware, como discos, terminais e periféricos. |
| `/opt`                 | Software opcional ou complementos instalados manualmente, frequentemente usados por gerenciadores como Homebrew. |
