# Números

## Descrição
Números são símbolos usados para representar quantidades, ordens e medidas. Eles são organizados em diferentes conjuntos, cada um com características específicas:

- **Números Naturais (ℕ):** São os números usados para contar objetos, começando geralmente em 0 ou 1 (0, 1, 2, 3, ...). São inteiros positivos sem frações.
- **Números Inteiros (ℤ):** Incluem todos os naturais, seus negativos e o zero (..., -3, -2, -1, 0, 1, 2, 3, ...). Usados para representar quantidades completas, podendo ser negativas.
- **Números Racionais (ℚ):** São números que podem ser expressos como a divisão de dois inteiros, onde o denominador é diferente de zero (ex: 1/2, -3/4, 5). Incluem números inteiros e frações exatas.
- **Números Irracionais:** Não podem ser expressos como fração de inteiros e têm uma representação decimal infinita e não periódica (ex: π, √2, e). São números reais, mas não racionais.
- **Números Reais (ℝ):** Compreendem todos os números racionais e irracionais, incluindo todos os valores que podem ser representados numa reta contínua.
- **Números Complexos (ℂ):** Incluem números que têm uma parte real e uma parte imaginária (ex: 3 + 4i). São usados para representar fenômenos que não podem ser descritos apenas pelos números reais.
- **Números Racionais Negativos:** São os números racionais menores que zero (ex: -1/2, -3).
- **Números Irracionais Transcendentais:** Um tipo especial de irracionais que não são solução de nenhuma equação polinomial com coeficientes inteiros (ex: π, e).
- **Números Algébricos:** Números que são solução de alguma equação polinomial com coeficientes inteiros (ex: √2 é algébrico, pois é solução de x² - 2 = 0).
- **Números P-adicos:** Um sistema numérico usado em teoria dos números, com propriedades diferentes dos números reais.
- **Números Ordinais e Cardinais:** Usados em matemática pura para tratar conceitos de ordem e quantidade infinita.
- **Números Hyperreais:** Utilizados em análise não padrão para formalizar o conceito de infinitesimais.
